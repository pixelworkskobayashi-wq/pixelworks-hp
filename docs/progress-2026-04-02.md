## 完成済み機能

### ヘッダー
- 固定表示（z-index: 100）
- PIXEL WORKS ロゴ（アクセントカラー）
- ナビ4項目：SERVICES・WORKSはトップページ内スクロール、ABOUT・CONTACTはページ遷移

### HERO
- フルスクリーン（100svh）
- Unsplash建築写真（後で実写真に差し替え可能）
- グラデーションオーバーレイ
- キャッチコピー＋サブテキスト（`ARCHITECTURAL VISUALIZATION — SAPPORO`）
- CTAリンク：`VIEW WORKS →` / `CONTACT`

### SERVICES
- 6サービスを3列グリッドで表示（レスポンシブ対応）
- 番号（01〜06）＋日本語タイトル＋英語サブタイトル
- スクロールフェードイン

### WORKS ← 最後に修正済み
- **作品数**: 24件（rendering/movie/vr/photo）
- **サムネ形状**: 横長（16:9）・縦長（3:4）・正方形（1:1）が混在
- **登場アニメーション**: IntersectionObserverで `.in-view` 付与
  - サムネ：opacity 0→1 ＋ translateY(24px)→0 / 0.7s
  - テキスト：opacity 0→1 ＋ translateX(12px)→0 / 0.5s（0.65s遅延）
- **ホバー**: scale(1.06) ズーム
- **カテゴリフィルター**: ALL / RENDERING / MOVIE / VR / PHOTO
- **モーダル**: クリックでその場拡大（ページ遷移なし）、ESCや外側クリックで閉じる

### ABOUT（/about）
- CONCEPT・会社情報・使用ツールセクション
- スクロールフェードイン

### CONTACT（/contact）
- Netlify Forms（サーバー不要）
- フィールド：お名前・会社名・メール・内容・添付ファイル
- 送信後：サンクスメッセージ表示

## 次にやること（未実装）

### モーダルの拡張機能（作業中断中）
以下の機能を実装予定だった：

1. **クリック時ズームインアニメーション** — 瞬間表示→スムーズなズームイン
2. **中間拡大モード**
   - 画像にホバーで虫眼鏡（`zoom-in`）カーソル
   - クリックでさらに拡大（最大拡大モード）
3. **最大拡大モード**
   - 余白の白背景を消す（フルスクリーン画像）
   - ホバーで手のひら（`grab`）カーソル → ドラッグ中は `grabbing`
   - 画像パン可能（慣性あり、弱め）
   - クリックで中間拡大に戻る
4. **余白クリックでギャラリーに戻る**（ズームアウトアニメーション付き）
5. **左右ドラッグで前後の作品に遷移**（中間拡大時）
