## 1. プロジェクト概要

### 会社情報
- **会社名**: Pixel Works（ピクセルワークス）
- **拠点**: 札幌（北海道）
- **業種**: 建築ビジュアライゼーション / 建築CGパース制作 / 写真撮影
- **ドメイン**: pixel-works.info

### 事業内容
1. 建築パース制作（外観・内観、フォトリアル高品質CG）
2. CGムービー制作（GPUレンダリングによる建築CGアニメーション）
3. 建築ビジュアライゼーション VR/インタラクティブ（TwinMotionクラウド）
4. 画像合成・加工（竣工写真へのCG合成）
5. 設計支援シミュレーション（イメージ共有・業務円滑化支援）
6. 竣工写真撮影

### サービスコンセプト
> 建築とグラフィックを結び、建築プロジェクトに関わるすべての方の業務円滑化・イメージ共有を支援する

### ターゲット
- 建築設計事務所・ゼネコン（BtoB）
- 不動産デベロッパー・ハウスメーカー（BtoB）

## 3. 技術スタック

| 項目 | 選定 | 理由 |
|------|------|------|
| フレームワーク | **Astro** | 静的サイト生成、表示速度最速クラス、ビジュアル表現に強い |
| ホスティング | **Netlify**（無料） | Netlify Forms が使える、GitHubと自動連携 |
| フォーム | **Netlify Forms** | サーバー不要、無料枠（月100件）で十分 |
| ドメイン | pixel-works.info（既存） | 追加費用ゼロ |
| コンテンツ更新 | Claude Codeと協働 | HTMLの知識不要、ファイルを直接編集 |

## 5. 各セクション仕様

### ヘッダー（全ページ共通・固定）
- ロゴ（PIXEL WORKS、英字大文字）
- ナビゲーション4項目（英字大文字）
- スクロールで背景が白くなるなどの subtle なエフェクト

### HERO（トップのみ）
- フルスクリーン（画面全体）の建築CGビジュアル
- キャッチコピー（日本語）
- サブテキスト：`ARCHITECTURAL VISUALIZATION — SAPPORO`
- CTAリンク：`VIEW WORKS →` / `CONTACT`
- **実装時**：サンプルとしてUnsplashの建築写真を使用。後から実画像に差し替え可能な設計。

### SERVICES（トップ内セクション）
- 6サービスを番号付きで並列表示
- 番号（01〜06）、日本語タイトル、英語サブタイトル、`LEARN MORE →` リンク
- 枠なし、余白で区切るミニマルなグリッドレイアウト

### WORKS（トップ内セクション）
- 作品グリッドギャラリー
- カテゴリフィルター（ALL / RENDERING / MOVIE / VR / PHOTO）
- **ホバーエフェクト**：画像が微妙にズーム（`transform: scale(1.05)`）、参考サイト（aesthetica.studio）に近い subtle な動き
- **クリック**：モーダル（ライトボックス）で拡大表示。ページ遷移なし。
- **実装時**：Unsplashの建築・インテリア写真をプレースホルダーとして使用

### ABOUT（/about）
- 会社理念・コンセプト
- 代表プロフィール
- 会社情報（所在地・設立など）
- 使用ソフトウェア・制作環境

### CONTACT（/contact）
- 問い合わせフォーム
  - お名前（必須）
  - 会社名
  - メールアドレス（必須）
  - お問い合わせ内容（必須）
  - 添付ファイル
- 送信処理：Netlify Forms
- 送信後：サンクスメッセージ表示

### フッター（全ページ共通）
- ロゴ・コピーライト
- ナビリンク（簡略版）
- SNSリンク（必要に応じて）

## 6. デザイントークン

### カラー
| 用途 | 値 | 備考 |
|------|-----|------|
| 背景 | `#ffffff` | 純白ベース |
| メインテキスト | `#1a1a1a` | ほぼ黒 |
| サブテキスト | `#888888` | グレー |
| 区切り線 | `#eeeeee` | 薄グレー |
| アクセント | **後日決定** | 単色。CSS変数 `--color-accent` で管理し全体に適用 |

### タイポグラフィ
- 見出し：英字大文字（`letter-spacing: 3px` 程度）
- 本文：日本語は読みやすいゴシック体
- ボタン・CTA：英字大文字、黒テキスト＋アンダーライン

### ボタン・CTA
- 塗りなし・枠なし
- 黒テキスト（`#1a1a1a`）+ ボトムボーダー（`1px solid #1a1a1a`）
- 英字大文字・文字間隔あり（例：`VIEW WORKS →`）
- ホバー時：opacity 変化または色変化

### アニメーション
- **スクロールフェードイン**：各セクションがスクロールで表示される際にフェードイン＋translateY
- **作品ホバー**：`transform: scale(1.05)` + `transition: 0.4s ease`
- **ナビホバー**：opacity またはアンダーラインのアニメーション

## 8. 制約・注意事項

- アクセントカラーはCSS変数（`--color-accent`）で一元管理し、後から1行変更で全体に反映できるようにする
- 作品画像は `src/assets/works/` 以下に置き、差し替えやすい構造にする
- コンテンツテキスト（サービス説明・会社概要等）はAstroのコンテンツコレクションまたはコンポーネントのpropsで管理し、HTMLの知識なしで更新できるようにする
- モバイル対応（レスポンシブ）必須
